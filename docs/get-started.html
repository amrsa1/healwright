Get Started

01 Installation

$ npm install healwright

02 Configuration

Set your API key and choose your provider (default is OpenAI).

# OpenAI (or AI_PROVIDER="gpt")
export AI_API_KEY="your-openai-key"
export SELF_HEAL=1

# Anthropic (or AI_PROVIDER="claude")
export AI_PROVIDER="anthropic"
export AI_API_KEY="your-anthropic-key"
export SELF_HEAL=1

# Google (or AI_PROVIDER="gemini")
export AI_PROVIDER="google"
export AI_API_KEY="your-google-key"
export SELF_HEAL=1

03 Setup Fixture

Create a fixture to wrap your Playwright page.

// fixtures.ts
import { test as base } from '@playwright/test';
import { createHealingFixture, HealPage } from 'healwright';

export const test = base.extend<{ page: HealPage }>(createHealingFixture());
export { expect } from '@playwright/test';

04 Write Your First Test

Use the wrapped page in your tests. If a selector breaks, AI fixes it automatically.

import { test, expect } from './fixtures';

test('login flow', async ({ page }) => {
  await page.goto('https://example.com');
  
  // If this selector breaks, AI will find the right element
  await page.heal.click(
    page.locator('[data-testid="submit-btn"]'),
    'Submit button on form'
  );
  
  // Or skip the selector entirely
  await page.heal.fill('', 'Email input field', 'user@example.com');
});

05 Available Methods

All healing methods take a locator (or empty string for AI-only) and a description.

Method What it does
heal.locator(selector, desc) Chainable self-healing locator
heal.click(loc, desc, {force?}) Click (force: true for hidden)
heal.fill(locator, desc, value) Fill an input
heal.selectOption(locator, desc, value) Select from dropdown
heal.check(locator, desc) Check a checkbox
heal.dblclick(locator, desc) Double-click
heal.hover(locator, desc) Hover over element
heal.focus(locator, desc) Focus element

06 AI-Only Mode

Pass an empty string as the locator and let AI find the element based on your description alone.

await page.heal.click('', 'Login button');
await page.heal.fill('', 'Search input', 'my query');

07 Cache Configuration

Healed selectors are cached in .self-heal/. Add it to your .gitignore: 

# .gitignore
.self-heal/
